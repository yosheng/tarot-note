小阿尔克纳-钱币
=======================================

.. image:: images/pentacle.gif

钱币代表元素(干冷):固执、务实、可掌握、稳重、可掌握、养成、融合

- 象征意义：富足与丰饶，母亲与守护，死亡与重生
- 力量：财富、成长、植物、出生、死亡、守护
- 型态：固态
- 方位：北方
- 季节：冬季
- 时刻：午夜
- 精灵：诺姆Gnome
- 天使：Uriel
- 圣兽：蜂鸟－（不过他是北方的守护者，其实不单纯是大地的象征物。）
- 颜色：黑色、褐色、绿色、白色
- 星座：金牛（本位）、魔羯（固定）、处女（转变）
- 行星：地球与土星、金星
- 净化象征：海盐（除去不洁）或水晶（净化与稳定能量）
- 象征物：五芒星、盐、大釜、绳、石头、水晶
- 植物：杉、柏、长春藤、橡树
- 动物：牛、鹿、狼、狗
- 女神：迪密特Demeter、帕瑟芬Persephone、盖亚Gaia、达奴Danu、伊希丝Isis
- 男神：瑟那诺斯Cernunnos、潘Pan、戴奥尼索斯Dionysus、奥西利斯Osiris、盖布Geb
