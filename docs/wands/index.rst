小阿尔克纳-权杖
=======================================

.. image:: images/fire.gif

权杖代表火元素(干热):破坏、热情、燃烧、绝望、重生、力量、侵略、温暖、光明、权力、威胁、希望、单纯、相互毁灭定律、牺牲、迅速、依赖、耀眼、不服输、魅力、神圣、保护、制裁、生命力、专注度、爆发力、阳性、温度、意志、血液

- 象征意义：热情与光明，意识与太阳，权力与真理
- 力量：改变、重生、激情、活力、荣耀与权力、净化与觉醒
- 型态：不定型
- 方位：南方
- 季节：夏季(也有人当作春季)
- 时刻：正午
- 精灵：萨拉曼德salamander
- 天使：Michael
- 圣兽：蛇
- 颜色：红色、橙色、金色、紫色
- 星座：牡羊（本位）、狮子（固定）、射手（转变）
- 行星：太阳与火星以及冥王星
- 净化象征：火、蜡烛(常结合色彩)、炭火与烟火
- 象征物：权杖、匕首、蜡烛
- 植物：辣椒、罗勒、肉桂、槿
- 动物：狮、蛇、狐、蝾螈
- 女神：布莉姬Brigit、芙依雅Freyia、维斯塔Vesta
- 男神：赫发斯特斯Hephaestus、拉Ra
