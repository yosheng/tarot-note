小阿尔克纳-圣杯
=======================================

.. image:: images/cup.gif

圣杯表代水元素(湿冷):柔软、调和、无形、浸透、倒印、假象、破坏、变化、纯洁、包容、感情、悲伤、沉淀、生命、流通、愉快、温柔、随和、净化、永恒、流逝、坚毅、亲合、交流

- 象征意义：直觉与情感，潜意识与月亮，情绪与净化
- 力量：梦、生命之源、情绪、爱、灵感、创造
- 型态：液态
- 方位：西方
- 季节：秋季(也有人当作夏季)
- 时刻：傍晚
- 精灵：温蒂妮Udine
- 天使：Gabriel
- 圣兽：美洲豹－（不过他是西方的守护者，其实不单纯是水的象征。）
- 颜色：蓝色、靛色、白色
- 星座：巨蟹（本位）、天蝎（固定）、双鱼（转变）
- 行星：月亮与海王星(也有人会加上冥王星)
- 净化象征：水（洗净）、精油或空间喷雾（不同配方将有不同效果）、花精
- 象征物：圣杯、镜、大釜、双螺旋纹
- 植物：水仙、百合、莲花、柳树
- 动物：鱼、蟹、蛙、天鹅
- 女神：阿芙萝黛蒂Aphrodite、艾波那Epona、拉恩Ran
- 男神：波塞顿Poseidon、奥克安诺斯Okeanos、里尔Lir、密米尔Mimir
