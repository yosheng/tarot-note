小阿尔克纳-宝剑
=======================================

.. image:: images/sword.gif

宝剑代表风元素(湿热):迅速、多变、理性、破坏、智慧、传递、锐利、无处不在、分散、冰冷、自由、快乐、冷静、疯狂、助燃、骄傲、升华、呼吸、生命、灵性、思维、逻辑、传导神经、神经系统、专业、传播、教授、沟通、研究、辅助者

- 象征意义：知性与思考，创造与语言，敏捷与专注
- 力量：破坏、传递、自由、抽象、创造与咒语
- 型态：气态
- 方位：东方
- 季节：春季(也有人当作秋季)
- 时刻：黎明
- 精灵：西尔芙Sylph
- 天使：Raphael
- 圣兽：鹰
- 颜色：蓝色、黄色、白色、紫色、灰色
- 星座：天平（本位）、水瓶（固定）、双子（转变）
- 行星：水星与天王星以及木星
- 净化象征：香、白色鼠尾草或松针、羽扇
- 象征物：刀、剑、香炉、香、羽毛、铃铛
- 植物：薰衣草、香茅、没药、松树
- 动物：鹰、蝴蝶、昆虫
- 女神：雅典那Athena、奴特Nut、希拉Hera
- 男神：舒Shu、赫密斯Hermes、奥丁Odin
